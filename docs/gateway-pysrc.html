  3  Module implementing gateway

 11  Exemple 
 12  ======= 
 13   
 14  >>> #Gateway code. 
 15  >>> gw = Gateway("effe") #Creating gateway named effe formated for 16 bytes long. 
 16  >>> gw.getName()  
 17  b'effe000000000000'  
 18  >>> gw.startLoRa() # Start LoRa service in the device. 
 19  >>> gw.addNewNode("f00a", b'NotSecureKey') #Add a device to the network. 
 20  >>> gw.sendMsg("Hello", "f00a") # Send message to device named f00a. 
 21  5 
 22   
 23  >>> # Node Code 
 24  >>> gw = Gateway("f00a") 
 25  >>> gw.addNewNode("effe", b'AnOtherSecureKey') # Add the gateway to known nodes.

 27  >>> gw.recvMsg() #message is saved in a FIFO. 
 28  True 
 29  >>> gw.popOldestMsg() # return a list with senders and message received. 
 30  [b'effe', b'Hello'] 
 31   
 32  Default configuration doesn't crypt your data. 
 33  See L{Gateway.sendMsg} for encryption.
